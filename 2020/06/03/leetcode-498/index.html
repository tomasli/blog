题目：
来自探索-二维数组简介

给定一个含有 M x N 个元素的矩阵（M 行，N 列），请以对角线遍历的顺序返回这个矩阵中的所有元素，对角线遍历如下图所示。

这个有点意思，练一练数组遍历的手。 图里就是先向右上的蛇形遍历
应该有个标志量来说明目前轮次是向右上还是向左下遍历
终结点必然是右下

class Solution(object):
    def findDiagonalOrder(self, matrix):
        """
        :type matrix: List[List[int]]
        :rtype: List[int]
        """
        if not matrix:
            return []

        # 奇数轮向右上遍历，偶数轮向左下遍历
        flag = 1

        res = []

        i, j = 0, 0

        while i <= len(matrix)-1 and j <= len(matrix[0])-1:
            res.append(matrix[i][j])
            # 之前向右上遍历的
            if flag%2:
                # 如果可以继续向右上走
                if i-1 >= 0 and j+1<=len(matrix[0])-1:
                    i -= 1
                    j += 1
                # 要转向了
                else:
                    flag += 1
                    # 右边还有元素，向右一格
                    if j+1 <= len(matrix[0])-1:
                        j += 1
                    # 右边没元素了，向下走
                    else:
                        i += 1


                # # 遍历到第一行了，肯定换方向
                # if i-1 < 0:
                #     flag += 1
                #     # 右边也没更多列了，就该向下走一格
                #     if j+1 > len(matrix[0])-1:
                #         i += 1
                #     # y右边还有元素，向右一格
                #     else:
                #         j += 1
                # # 还可以继续向右上走
                # else:
                #     # 但是右边已经没列了，向下一格，也换方向
                #     if j+1 > len(matrix[0])-1:
                #         flag += 1
                #         i += 1
                #     # 真正可以向右上走
                #     else:
                #         i -= 1
                #         j += 1
            else:
                # 可以继续向左下走
                if j-1 >=0  and i+1<=len(matrix)-1:
                    i += 1
                    j -= 1
                # 要转向
                else:
                    flag += 1
                    # 下方还有元素，向下一格
                    if i+1<=len(matrix)-1:
                        i += 1
                    # 必须向右了
                    else:
                        j += 1

        return res
