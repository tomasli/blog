题目：

实现 int sqrt(int x) 函数。

计算并返回 x 的平方根，其中 x 是非负整数。

由于返回类型是整数，结果只保留整数的部分，小数部分将被舍去。

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/sqrtx

写了3种方法

  1. 单步加1速度太慢, 写的时候根本没想效率的事

    class Solution(object):
    def mySqrt(self, x):
        """
        :type x: int
        :rtype: int
        """
        y = 0
        while y*y <= x:
            y+=1
        return y-1

  2. 二分查找, 自己想了一大顿关联到了，写的过于繁琐，设置四种判断条件人为变得困难

    class Solution(object):
    def mySqrt(self, x):
        """
        :type x: int
        :rtype: int
        """
        # 临界条件设置的好傻
        if x == 0:
            return 0
        if x == 1:
            return 1
    
        left, right = 0, x 
        while left <= right:
            # 取整应该用//
            mid = left + (right - left)/2
            # 指数是**，不是^（按二进制位异或运算）
            if mid**2 <= x and (mid+1)**2 > x:
                return mid
            elif mid**2 <= x and (mid+1)**2 <= x:
                left = mid
            elif mid**2 > x and (mid-1)**2 <= x:
                # 返还忘了减1
                return mid-1
            elif mid**2 > x and (mid-1)**2 > x:
                right = mid

  3. 简单版二分查找，正确答案默写

    class Solution(object):
    def mySqrt(self, x):
        """
        :type x: int
        :rtype: int
        """
        l, r, res = 0, x, -1

        while l <= r:
            mid = l + (r - l)/2
            if mid**2 <= x:
                res = mid
                l = mid + 1
            elif mid**2 > x:
                r = mid - 1
        # 保存唯一正确位置到最后返回就行
        return res
