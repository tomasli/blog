题目：

实现 pow(x, n) ，即计算 x 的 n 次幂函数。

前边都是错误解法，想破头。后边才是正确的

###错误解法1

class Solution(object):
    def myPow(self, x, n):
        """
        :type x: float
        :type n: int
        :rtype: float
        """

        def helper(x, n):
            if n == 0:
                return 1
            elif n > 0:
                return helper(x, n/2)*helper(x, (n-n/2))
            elif n < 0:
                return helper(x, n/2)/helper(x, (n-n/2))

        return helper(x, n)

首先想到动态规划，myPow(x, n)=myPow(x, n-m)*myPow(x, m)，状态转换
myPow(x, 0) = 1, 这个base case，不论正负值都往这个方向走
用迭代试试，分成>0, <0两部分
以为写的还行呢，其实是终结条件没判断好，n=1的时候，两个相乘的helper会总有一个以helper(x, 1)深入进去，达到迭代最大值

###以下正确答案

参考了别人答案才会了，可以变形：x的n次方=(1/x)的-n次方，用这个把负数n变正数。另外本不需要helper方法

class Solution(object):
    def myPow(self, x, n):
        """
        :type x: float
        :type n: int
        :rtype: float
        """
        if n == 0:
            return 1.0

        if n < 0:
            x = 1/x
            n = -n

        if n%2 == 0:
            return self.myPow(x, n/2)**2
        elif n%2 == 1:
            return self.myPow(x, n/2)**2*x

这块还可以优化，进入子迭代前平方，而不要返回时平方，更易读，效率上差不多

class Solution(object):
    def myPow(self, x, n):
        """
        :type x: float
        :type n: int
        :rtype: float
        """
        if n == 0:
            return 1.0

        if n < 0:
            x = 1/x
            n = -n

        if n%2 == 0:
            return self.myPow(x**2, n/2)
        elif n%2 == 1:
            return self.myPow(x**2, n/2)*x
